Title : Download manager animation is displayed improperly on Windows and Ubuntu
Keywords :display,animation,windows,ubuntu,download,manager,
Assigned Developer : randomDev4

Solved status : open
Bug : User Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/62.0.3202.94 Safari/537.36


Step to reproduce:
Launch Firefox and navigate to http://testsafebrowsing.appspot.com/
Go to Desktop Download Warnings
Right click the "link" in order to trigger the context menu.
Select "Save link as" option and save it locally.

Actual results:
Once the download animation ends, the beginning of the animation is displayed once again.

Expected results:
Once the download starts, the animation is displayed only once, without any issues.
