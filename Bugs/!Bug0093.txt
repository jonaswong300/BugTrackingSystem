Reproduced on Latest Nightly, build ID: 20141202030201 on a Microsoft Surface Pro 2 device running Windows 8.1 64-bit.


Step to reproduce:

1. Open Nightly using a clean profile in landscape mode;
2. Enter any term in the new Search bar;
3. Put the device in portrait mode.
Actual results: The dropdown is misplaced after changing the device orientation.

Expected results: Dropdown should change properly with the orientation.
